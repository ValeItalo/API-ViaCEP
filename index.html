WebService de CEP

Eu desenvolvi a aplicação para simular uma página de cadastro. A busca do endereço é realizada automaticamente após o usuário digitar o CEP. Implementei validações para impedir a digitação de um CEP inexistente, além da verificação dos campos ao realizar o submit do formulário.

Alguns dos elementos utilizados:

  • API: viaCEP
  • Elemento Dialog
  • Keyup Event
